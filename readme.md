# MM-Rando-Tracker

A web based tracker for MM Randomizer.

WIP and may never be done.

## Goals

- Item tracker
- Location checklist

- Upload logic file

## Development

Written in typescript, and uses `ts-watch`. Install npm dependencies and execute:

```
npm run dev
```
